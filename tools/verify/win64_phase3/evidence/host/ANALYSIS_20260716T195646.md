## False PASS root cause

Host `.cmd` scripts did:

```bat
dalvikvm.exe ...
echo exit=%ERRORLEVEL%>> log
type log
echo exit=%ERRORLEVEL%
```

`type` resets `%ERRORLEVEL%` to 0, so `run_all_host.cmd` saw success even when dalvikvm failed. Fixed by capturing `set RC=%ERRORLEVEL%` before `type` and `exit /b %RC%`.

## Product root cause

`Java_libcore_io_Linux_poll` used `WSAPoll` on CRT `_open_osfhandle` sockets. Works under wine64; **real Win10 returns WSAEINVAL** on accept-path poll. Fixed by implementing poll/read-timeout/write-wait with **`select()`**.
